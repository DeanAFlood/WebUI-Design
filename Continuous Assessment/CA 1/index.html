Web Protocols

HTTP (Hypertext Transfer Protocol) is the foundation of data communication on the web. It defines the rules for requests and transfers

HTTPS(Hypertext Transfer Protocol Secure) Is a secure verion of HTTP, it uses encryption to protect data exchange

TCP/IP protocol suite takes place when data is sent over the internet,this protocol handles the transference of data between your computer and its Applications to the network and other computers

TCP (Transmission Control Protocol) takes care of communinication between your application, Browsers for example, and your network and breaks them down into IP packets

IP (Internet Protocol) is responsible for routing each IP packet to its destination (communication between computers)

FTP (File transfer Protocol) enables the transfer of files
